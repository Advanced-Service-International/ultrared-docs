## 🚀 Despliegue

Todas las páginas están alojadas mediante **GitHub Pages** y se actualizan automáticamente con cada commit a `main`.

**URL base:**
```
https://advanced-service-international.github.io/ultrared-docs/
```

---

## ✏️ Cómo actualizar una página

1. Navega al archivo dentro de `docs/`
2. Haz click en el ícono ✏️ de edición
3. Realiza tus cambios
4. Haz commit directamente a `main`

GitHub Pages se actualiza automáticamente en 1–2 minutos.

---

## ➕ Cómo agregar una nueva app

1. Crea una carpeta con el nombre de la app dentro de `docs/`
2. Agrega los archivos HTML necesarios (`support.html`, `privacy.html`, etc.)
3. Actualiza la tabla de aplicaciones en este README
4. Haz commit a `main`
